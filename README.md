# Conexciencia

Sitio web one-page de **Conexciencia**, un proyecto de divulgación científica orientado a escuelas, organizaciones y comunidades. Su lema es: *«¡No es magia, es ciencia!»*.

Este repositorio parte de una plantilla HTML multipropósito (Assan / Bootstrap) adaptada y personalizada con el contenido, identidad visual e imágenes de Conexciencia.

---

## Descripción

La página presenta la propuesta de Conexciencia: acercar la ciencia de forma lúdica y experimental a través de talleres, presentaciones y laboratorios. Incluye información institucional, servicios, planes con precios, testimonios, preguntas frecuentes y un formulario de contacto.

**Sitio principal:** [`html/index.html`](html/index.html)

---

## Secciones del sitio

| Sección | ID | Contenido |
|---------|-----|-----------|
| Inicio | `#home` | Hero con parallax y mensaje principal |
| Nosotros | `#about` | Sobre nosotros, misión y visión |
| Servicios | `#services` | Qué ofrece Conexciencia (experimentos, juego, exploración, aprendizaje) |
| Planes | `#pricing` | Tres opciones: Opción 1, Opción 2 y Laboratorios de Asombro |
| Experiencias | `#testimonials` | Testimonios / experiencias |
| Preguntas frecuentes | `#faq` | Dudas comunes sobre actividades y reservas |
| Contáctanos | `#contact` | Datos de contacto y formulario |

---

## Tecnologías

- **HTML5 / CSS3 / JavaScript**
- **Bootstrap 4** (layout y componentes)
- **jQuery** y plugins asociados
- **Smart Forms** — formulario de contacto con validación
- **DZS Parallaxer** — hero con parallax
- **WOW.js / Animate.css** — animaciones al hacer scroll
- **Font Awesome** e iconos ET Line

No requiere ningún framework de frontend, ya que es un sitio estático listo para servir desde cualquier hosting o servidor local.

---

## Cómo ver el proyecto en local

1. Abre la carpeta `html/` en un servidor local o abre `index.html` directamente en el navegador.
2. Con VS Code puedes usar la extensión **Live Server**.


> **Nota:** Algunas funciones del formulario (envío / reCAPTCHA) pueden requerir configuración en el servidor o en los archivos PHP dentro de `smart-form/contact-recaptcha/`.
